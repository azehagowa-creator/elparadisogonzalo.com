# Elparadisogonzalo.com

A modern Web3 platform built with Vite, React, Capacitor, and blockchain technologies. The project includes a responsive website, Android application, and APIs with support for Ethereum, BNB Smart Chain, and decentralized integrations.

## Features

- Web3-enabled dApp
- Responsive React + Vite frontend
- Android app powered by Capacitor
- Ethereum and BNB Smart Chain support
- GitHub Pages and custom domain deployment
- Unstoppable Domains integration
- Modern UI with Framer Motion
- React Router navigation

## Requirements

- Node.js 22+
- npm 10+

## Installation

```bash
git clone https://github.com/azehagowa-creator/elparadisogonzalo.com.git
cd elparadisogonzalo.com
npm install
```

## Development

```bash
npm run dev
```

## Build

```bash
npm run build
```

## Preview

```bash
npm run preview
```

## Android

Synchronize Capacitor:

```bash
npm run sync
```

Open Android Studio:

```bash
npm run android
```

Build Debug APK:

```bash
npm run apk
```

Build Release APK:

```bash
npm run release
```

Generate Android App Bundle:

```bash
npm run bundle
```

## Lint
